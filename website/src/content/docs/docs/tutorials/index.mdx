Tutorials are hands-on lessons. Use this section when you want to learn a workflow end to end before turning it into an operating runbook.

## What is in this section

1. [Observe Bub with tapes and Jaeger](/docs/tutorials/observability/) — inspect Bub's own tape first, then export Logfire/OpenTelemetry traces to Jaeger.
2. [Connect MCP Servers with bub-mcp](/docs/tutorials/mcp/) — install the MCP plugin, wire up a time server, and call it from a Bub turn.
3. [Persist tapes in SQLAlchemy with local SQLite or Turso](/docs/tutorials/tapestore-sqlalchemy/) — replace the file-based tape store with a SQLAlchemy database, then push the same workflow to a Turso embedded replica.
