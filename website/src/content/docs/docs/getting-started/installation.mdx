---
title: Installation
description: Install Bub from PyPI or clone the repo and run it from source.
---

## From PyPI

```bash
pip install bub
```

## From source

```bash
git clone https://github.com/bubbuild/bub.git
cd bub
uv sync
uv run bub chat
```

## Core commands

- `uv run bub chat` starts the interactive CLI.
- `uv run bub run "summarize this repo"` runs a single turn through the full pipeline.
- `uv run bub gateway` starts channel listener mode.
