# Socialized Evaluation

Bub started in group chats, where agents were expected to coexist with people and other agents instead of owning the whole conversation.

That pressure led to a few durable choices:

- hooks instead of one-off extension seams
- tape as context storage instead of permanent transcript accumulation
- one runtime path across CLI and channel adapters
