---
title: 概念
description: 认识 hooks、tape 与跨 channel 运行时流程。
---

Bub 围绕一条统一的 turn pipeline 组织：

```text
resolve_session -> load_state -> build_prompt -> run_model
                                                   |
             dispatch_outbound <- render_outbound <- save_state
```

每一步都是 hook，builtins 只是默认插件实现。
